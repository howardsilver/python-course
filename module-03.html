Hello, World
Classic first program
Every programmer's very first program is Hello, World. It is a tradition going back to 1974. The code below already works — press Run to see it execute. Then try changing the text inside the quotes to anything you want and run it again.

The print() function is your program's way of talking back to you. Whatever you put inside the parentheses — wrapped in quotes — appears as output. That's it. Simple, but fundamental.

1
Output
The quotes tell Python "this is text, not a command." Without quotes, Python would try to find a variable called Hello and fail. Text wrapped in quotes is called a string — you'll learn much more about strings in Module 5.
It ran! Now type this in VS Code too — create a file called hello.py, paste the same code, save it with Ctrl+S, and press the play button.
