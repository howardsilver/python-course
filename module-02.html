Open your project folder
In VS Code, always open a folder, not individual files. This tells VS Code "this is my project". Click File → Open Folder, navigate to your python-course folder on your Desktop, and click Select Folder. You'll see it appear in the Explorer panel on the left.
~3 min Why open a folder instead of a file?
Opening a folder lets VS Code understand your whole project at once. It can find related files, run the right Python interpreter, and give you better autocomplete. Opening a single file is like reading one page of a book with no context — VS Code works much better when it sees the whole picture.
Create and edit a Python file
In the Explorer panel, hover over your folder name and click the New File icon (a page with a + sign). Name it practice.py and press Enter. It opens in the editor. Type something — anything — and press Ctrl+S to save. Notice the dot on the tab disappears when saved.
~5 min What does the dot on the tab mean?
A dot (●) on a file tab means there are unsaved changes. VS Code will never lose your work — it keeps track — but Python can only run the saved version of your file. Get into the habit of pressing Ctrl+S before you run anything.
Run a Python file — two ways
With hello.py open, try both methods:
Method 1 — Play button: Look for the triangle ▶ in the top-right corner of VS Code. Click it. A terminal opens at the bottom and your output appears. This is the easiest way.
Method 2 — Terminal: Press Ctrl+` (the backtick key, top-left of keyboard) to open the terminal. Type python hello.py and press Enter. Same result, more control.
~5 min Which method should I use?
Use the play button for quick runs while learning. Use the terminal when you get to more advanced work — it gives you more control and matches how professional developers run code. For now, either is fine.
Three shortcuts worth memorising now
These three keyboard shortcuts will save you time every single day. Practise each one right now while VS Code is open.
Ctrl+S — Save the current file. Do this constantly. Every time before you run.
Ctrl+Z — Undo the last thing you typed. Works up to hundreds of steps back.
Ctrl+` — Open or close the terminal. The backtick key is top-left, under Escape.
