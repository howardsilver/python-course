What is Python — and why start here?
Python is a programming language — a precise set of instructions that a computer can follow. It reads almost like plain English, which makes it ideal for beginners. Python is used everywhere that matters: Instagram, YouTube, NASA, Netflix, and virtually every AI and machine learning system you've heard of. It's also the dominant language for AI development — which is exactly where this course is heading. Starting with Python means the skills you build here transfer directly into the advanced topics that come later. It's the right call.
~3 min
Download Python 3 from python.org
Open your browser and go to python.org/downloads. Click the large yellow "Download Python 3.x.x" button — it automatically detects Windows and gives you the right version. Download the .exe installer file. This takes 1–2 minutes depending on your connection speed.
~4 min Which version should I pick?
Always use the latest Python 3 release — the yellow button on python.org gives you this automatically. Avoid Python 2 entirely; it's old, unsupported, and incompatible with modern code. Don't worry about minor version numbers like 3.11 vs 3.12 — any recent Python 3 version is fine.
Install Python — one critical checkbox
Run the installer you just downloaded. Before clicking anything else, look at the bottom of the first screen.
▲ Check "Add Python to PATH" first. This checkbox is at the bottom of the installer screen and is easy to miss. Without it, your terminal will not be able to find Python and nothing will work. Check it, then click "Install Now".
Wait for installation to complete (about 1–2 minutes), then click Close.
~6 min What does "Add to PATH" mean?
PATH is a list your computer keeps of "where to look for programs." When you type python in the terminal, Windows searches that list to find the Python program. Without Python in PATH, Windows has no idea it exists and every command will fail with a "not recognized" error.
Verify Python installed correctly
Let's confirm everything worked. Press Win + R, type cmd, press Enter. A black terminal window opens. Type the command below and press Enter.
python --version
# Expected output: Python 3.x.x
If you see a version number starting with 3 — Python is installed correctly. You can close the terminal window.
~3 min "python is not recognized" — how to fix it
This error means Python was not added to PATH during installation. The fix: go to Windows Settings → Apps → Installed Apps → find Python → Modify → check "Add Python to PATH" → Modify. Alternatively, uninstall Python and reinstall it — this time checking the PATH box before clicking Install Now.
Download and install VS Code
VS Code is the editor where you will write all your Python programs. It is free, fast, and used by professional developers worldwide. You will get a complete tour of it in Module 2 — for now, just get it installed.

Go to code.visualstudio.com and click the blue "Download for Windows" button. Run the installer, accept the defaults, and click Next through each screen. When VS Code opens automatically after installation, you are done.
~7 min Why VS Code and not Notepad?
Notepad is a plain text editor — it has no idea what Python code looks like. VS Code is a code editor built for programming. It highlights your code in color so it's easier to read, underlines mistakes as you type, autocompletes common patterns, and has a built-in terminal so you can run your programs without switching windows. Writing Python in Notepad is like trying to write a novel in a calculator.
