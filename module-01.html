What is Python? (5-min read)
Python is a programming language — a way to give instructions to a computer. It reads like plain English, runs on everything, and powers Instagram, YouTube, NASA, and almost every AI system you've heard of. It's the best first language, period.
~5 min Why Python and not another language?
Python is beginner-friendly, but also the #1 language for AI and machine learning — so what you learn here directly leads to building AI tools later. JavaScript is also popular, but it's messier and web-focused. Python is clean, readable, and has the best libraries for data, AI, and automation.
Download Python 3 from python.org
Go to python.org/downloads and click the big yellow button. It automatically detects Windows and gives you the right version. Download the .exe installer file.
~5 min Which version should I pick?
Always pick the latest stable Python 3 release — the big yellow button on python.org gives you this automatically. Avoid Python 2; it's old and unsupported. Don't worry about minor version numbers — 3.11, 3.12, 3.13 are all fine.
Install Python — the one critical checkbox
Run the installer. Before clicking anything else — check the box that says "Add Python to PATH". This is the single most common beginner mistake. Without it, your terminal won't know where Python is. Then click "Install Now".
▲ Don't skip this: The PATH checkbox is at the bottom of the installer screen. It's easy to miss. Scroll down and check it before doing anything else.
~5 min What does PATH mean?
PATH is a list your computer keeps of "where to look for programs". When you type python in the terminal, Windows searches PATH to find the Python program. Without it, Windows doesn't know Python exists.
Verify Python installed correctly
Press Win + R, type cmd, press Enter. A black window appears — this is the terminal. Type the command below and press Enter.
python --version
# You should see: Python 3.x.x
~3 min "python is not recognized" — how to fix it
This error means Python wasn't added to PATH. Fix: open Windows Settings → Apps → Installed Apps → find Python → Modify → check "Add Python to PATH" → Modify. Or uninstall Python and reinstall, this time checking that box.
Download and install VS Code
Go to code.visualstudio.com and download the Windows installer. VS Code is your code editor — where you'll write every program in this course. It's free, fast, and used by professional developers worldwide.
~5 min Why VS Code and not Notepad?
Notepad is a basic text editor. VS Code is a code editor — it highlights your code in colour so you can read it, underlines mistakes as you type, autocompletes your code, and has a built-in terminal. It turns a wall of text into something you can navigate.
Install the Python extension in VS Code
Open VS Code. Click the Extensions icon in the left sidebar (four squares). Search "Python" and install the one made by Microsoft (it has millions of downloads). This gives VS Code Python superpowers: colour highlighting, error detection, and a run button.
~5 min What does this extension actually do?
Without the extension, VS Code treats Python files like plain text. With it: keywords light up in colour, typos get underlined in red as you type, you get autocomplete suggestions, and you can run your program by pressing the green play button — no terminal needed.
Write and run your very first program
In VS Code: File → New File. Save it as hello.py inside a new folder called python-course on your desktop. Type the code below exactly. Press the ▶ play button in the top right corner. If you see your message printed in the terminal below — you're a programmer.
# My first Python program
print("Hello! My coding journey starts now.")
print("This is going to be awesome.")
~10 min What does print() actually do?
print() is a built-in Python function. A function is a named action your program can perform. print() specifically takes whatever you put inside the brackets and displays it in the terminal. It's how programs communicate with the person using them. You'll use it constantly.
